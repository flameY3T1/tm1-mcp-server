# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1. The full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

## Our pledge

We as members, contributors, and maintainers pledge to make participation in
this project a respectful, harassment-free experience for everyone, and to act
in ways that build an open and welcoming community.

## Scope

This Code of Conduct applies within all project spaces (issues, pull requests,
discussions) and when an individual is representing the project in public spaces.

## Reporting

Report unacceptable behavior privately via the channel described in
[SECURITY.md](SECURITY.md), or by opening a confidential report through GitHub.
All reports will be reviewed and handled discreetly.

Maintainers are responsible for clarifying standards and may take appropriate
and fair corrective action in response to any behavior they deem inappropriate.
For the full enforcement guidelines, see the Contributor Covenant link above.
